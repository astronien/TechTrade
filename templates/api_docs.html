รายการ Trade (V2)

ดึงรายการ Trade-In ทั้งหมดที่ถูกเก็บไว้ใน Turso Database (Cache) โดยเลือกดูรายโซนได้

GET /api/v2/trades

พารามิเตอร์ (Query Parameters)

ชื่อ ประเภท คำอธิบาย
zone string ชื่อโซนที่ต้องการกรอง (เช่น "BKK-1")
limit integer จำนวนรายการสูงสุด (Default: 100)

ตัวอย่างผลลัพธ์ (Response)

{
  "success": true,
  "count": 2,
  "data": [
    {
      "trade_in_id": "T6705001",
      "real_branch_id": "645",
      "document_no": "IV6705001",
      "document_date": "2026-01-15",
      "net_price": 12500.0,
      "series": "iPhone 15 Pro Max",
      "status": 4
    }
  ]
}

สถิติรายโซน

ดึงข้อมูลสรุปภาพรวมยอดเทรดแยกตามโซน เหมาะสำหรับการทำ Dashboard

GET /api/v2/stats

ตัวอย่างผลลัพธ์ (Response)

{
  "success": true,
  "stats": [
    {
      "zone_name": "BKK-1",
      "total_trades": 150,
      "total_value": 1850000.0,
      "last_updated": "2026-05-03"
    }
  ]
}

ตัวอย่างโค้ด (Python)

import requests

url = "https://your-domain.com/api/v2/trades"
params = {
    "zone": "BKK-1",
    "limit": 50
}

response = requests.get(url, params=params)
data = response.json()

if data['success']:
    print(f"พบข้อมูล {data['count']} รายการ")
    for trade in data['data']:
        print(trade['series'], trade['net_price'])

ตัวอย่างโค้ด (JavaScript)

async function fetchStats() {
  const response = await fetch('/api/v2/stats');
  const data = await response.json();
  
  if (data.success) {
    console.table(data.stats);
  }
}

fetchStats();
